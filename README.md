# 🎬 FilmVault – Movie Web App

A modern movie browsing web application built using React that allows users to explore, search, and manage their favorite movies using the TMDB API.

---

## 🚀 Features

- 🔍 Search movies in real-time
- 🎞 Browse trending/popular movies
- 📄 Pagination for smooth navigation
- ❤️ Add/Remove movies to Watchlist
- 🔽 Sort movies by rating
- 🎯 Filter movies by genre
- ⚡ Fast and responsive UI

---

## 🛠 Tech Stack

- React.js
- JavaScript (ES6+)
- Tailwind CSS
- Vite
- TMDB API
